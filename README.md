# twizoo-test

## Setup and Usage

### Install a virtual environment
```
$ pip install virtualenv
```

### Create the virtualenv
```
$ virtualenv flask
```

### Navigate to flask/ and enter
```
$ . bin/activate
```

### Navigate to app/ and enter
```
$ pip install -r requirements
```

### Start a mongoDB server instance
```
$ mongod
```

### In root folder type:
```
$ chmod a+x run.py
$ ./run.py
```
